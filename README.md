# NBP Lando | Official Artist Website

A modern, responsive music artist showcase website built with HTML, CSS, and JavaScript.

## Features

- **Hero Section** - Stunning introduction with animated background
- **Music Player** - Interactive track list with play/pause functionality
- **Social Links** - Connect on TikTok, Instagram, YouTube, Spotify
- **Contact Form** - Booking and collaboration inquiries
- **Responsive Design** - Works perfectly on all devices
- **Dark Theme** - Modern aesthetic with gradient accents

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- Font Awesome icons
- CSS gradients and animations
- Smooth scroll navigation

## Deployment

This site is deployed on Vercel: https://nbp-lando-music.vercel.app

## Customization

To personalize this site for NBP Lando:

1. **Replace Placeholder Content**
   - Update artist name and bio in hero section
   - Add real music tracks with audio files or embed links
   - Update social media handles and links

2. **Add Music**
   - Replace track list with real tracks
   - Add audio files or YouTube/SoundCloud embeds
   - Update track metadata (title, duration)

3. **Social Links**
   - Replace `@yourhandle` with actual social handles
   - Update links to real social media profiles

4. **Images**
   - Add artist photos
   - Include album artwork
   - Add background images or videos
